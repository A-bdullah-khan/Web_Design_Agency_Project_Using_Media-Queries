DESCRIPTION:
A modern design agency webpage built using HTML and CSS. This project showcases a creative agency layout with images, headings, and descriptive sections to highlight services.

FEATURES:
1) Logo and main heading section
2) Two service cards
3) Images with descriptive content
4) Clean and structured layout
5) Footer with tagline
6) Custom Google Fonts integration

CONCEPTS USED:
1) HTML Document Structure
2) Div Elements
3) Headings 
4) Paragraph Tag
5) Image Tag
6) CSS Styling
7) Class Selectors
8) Layout Structuring
9) Typography and Fonts
